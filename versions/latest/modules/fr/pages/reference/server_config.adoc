= Référence de configuration du serveur
:page-languages: [en, zh]
:revdate: 2026-02-27
:page-revdate: {revdate}

Ceci est une référence à tous les paramètres pouvant être utilisés pour configurer le serveur rke2. Notez que bien que ceci soit une référence aux arguments de ligne de commande, la meilleure façon de configurer RKE2 est d'utiliser le xref:install/configuration.adoc#_configuration_file[fichier de configuration].

== Valeurs de configuration critiques

Les options suivantes doivent être définies avec la même valeur sur tous les serveurs du cluster. Le non-respect de cette consigne empêchera les nouveaux serveurs de rejoindre le cluster.

* `agent-token`
* `cluster-cidr`
* `cluster-dns`
* `cluster-domain`
* `disable-cloud-controller`
* `disable-kube-proxy`
* `egress-selector-mode`
* `service-cidr`
