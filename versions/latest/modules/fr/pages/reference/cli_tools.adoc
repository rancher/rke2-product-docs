= Outils en ligne de commande (CLI)
:page-languages: [en, zh]
:revdate: 2025-08-21
:page-revdate: {revdate}

RKE2 expédie plusieurs outils CLI pour aider à accéder et à déboguer le cluster. Au démarrage, ils sont extraits dans `/var/lib/rancher/rke2/bin`.

== kubectl

Un kubeconfig administrateur est généré dans `/etc/rancher/rke2/rke2.yaml`.

Exemple :

[,bash]
----
export KUBECONFIG=/etc/rancher/rke2/rke2.yaml
/var/lib/rancher/rke2/bin/kubectl get nodes
----

== Containerd

RKE2 est livré avec `ctr` et `crictl`. Le socket Containerd est situé dans `/run/k3s/containerd/containerd.sock`.

Exemples :

[,bash]
----
/var/lib/rancher/rke2/bin/ctr --address /run/k3s/containerd/containerd.sock --namespace k8s.io container ls
----

[,bash]
----
export CRI_CONFIG_FILE=/var/lib/rancher/rke2/agent/etc/crictl.yaml
/var/lib/rancher/rke2/bin/crictl ps
----
