= Herramientas CLI
:page-languages: [en, zh]
:revdate: 21/08/2025
:page-revdate: {revdate}

RKE2 envía varias herramientas CLI para ayudar a acceder y depurar el clúster. Al iniciar, se extraen a `/var/lib/rancher/rke2/bin`.

== kubectl

Se genera un kubeconfig de administrador en `/etc/rancher/rke2/rke2.yaml`.

Ejemplo:

[,bash]
----
export KUBECONFIG=/etc/rancher/rke2/rke2.yaml
/var/lib/rancher/rke2/bin/kubectl get nodes
----

== Containerd

RKE2 se envía con `ctr` y `crictl`. El socket de Containerd se encuentra en `/run/k3s/containerd/containerd.sock`.

Ejemplos:

[,bash]
----
/var/lib/rancher/rke2/bin/ctr --address /run/k3s/containerd/containerd.sock --namespace k8s.io container ls
----

[,bash]
----
export CRI_CONFIG_FILE=/var/lib/rancher/rke2/agent/etc/crictl.yaml
/var/lib/rancher/rke2/bin/crictl ps
----
