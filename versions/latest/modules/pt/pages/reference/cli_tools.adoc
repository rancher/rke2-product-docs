= Ferramentas CLI
:page-languages: [en, zh]
:revdate: 2025-08-21
:page-revdate: {revdate}

O RKE2 fornece várias ferramentas CLI para ajudar no acesso e na depuração do cluster. Na inicialização, elas são extraídas para `/var/lib/rancher/rke2/bin`.

== kubectl

Um kubeconfig de administrador é gerado em `/etc/rancher/rke2/rke2.yaml`.

Exemplo:

[,bash]
----
export KUBECONFIG=/etc/rancher/rke2/rke2.yaml
/var/lib/rancher/rke2/bin/kubectl get nodes
----

== Containerd

O RKE2 vem com `ctr` e `crictl`. O socket do Containerd está localizado em `/run/k3s/containerd/containerd.sock`.

Exemplos:

[,bash]
----
/var/lib/rancher/rke2/bin/ctr --address /run/k3s/containerd/containerd.sock --namespace k8s.io container ls
----

[,bash]
----
export CRI_CONFIG_FILE=/var/lib/rancher/rke2/agent/etc/crictl.yaml
/var/lib/rancher/rke2/bin/crictl ps
----
