= Server-Konfigurationsreferenz
:page-languages: [en, zh]
:revdate: 2026-02-27
:page-revdate: {revdate}

Dies ist eine Referenz zu allen Parametern, die zur Konfiguration des rke2-Servers verwendet werden können. Bitte beachten Sie, dass dies eine Referenz zu den Befehlszeilenargumenten ist – der beste Weg, RKE2 zu konfigurieren, besteht jedoch darin, die xref:install/configuration.adoc#_configuration_file[Konfigurationsdatei] zu verwenden.

== Kritische Konfigurationswerte

Die folgenden Optionen müssen auf allen Servern im Cluster auf denselben Wert gesetzt werden. Das Versäumnis, dies zu tun, wird dazu führen, dass neue Server dem Cluster nicht beitreten können.

* `agent-token`
* `cluster-cidr`
* `cluster-dns`
* `cluster-domain`
* `disable-cloud-controller`
* `disable-kube-proxy`
* `egress-selector-mode`
* `service-cidr`
